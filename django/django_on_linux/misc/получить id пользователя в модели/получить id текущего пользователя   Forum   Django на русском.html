"Если очень хочется, но нельзя, значит можно" - глупая поговорка

Для того, чтобы в файле с моделями получить доступ к объекту request, необходимо создать middleware (как это сделать найдете по этой ссылке: http://djbook.ru/rel1.6/topics/http/middleware.html) с таким содержимым:

# coding=utf-8
import threading

_thread_locals = threading.local()

def get_request():
    return getattr(_thread_locals, 'request', None)

class ThreadLocals(object):
    """
    Middleware, который получает объект request
    и делает его доступным глобально
    """
    def process_request(self, request):
        _thread_locals.request = request

В файле с моделями добавляем такой код:

from middleware.threadlocals import get_request
request = get_request()

Из request можно извлечь id usera. (как сделать смотрите тут: http://djbook.ru/rel1.6/ref/request-response.html
