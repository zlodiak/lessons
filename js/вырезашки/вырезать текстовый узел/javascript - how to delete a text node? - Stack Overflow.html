    Your "Бесплатно!" text is not in an element by itself. It is under the li element, and removing that element would not be desirable I imagine. You could manipulate its innerHTML to remove that text.

    var li = document.querySelectorAll('li');
for (i = 0; i < li.length; i++) {
  li[i].innerHTML = li[i].innerHTML.replace("Бесплатно!", "");
}

    If you're looking to delete every element containing "Бесплатно!" using jQuery (is this what you mean by text node?) then you can use the jQuery :contains selector to find these elements, and remove them:

    $("li:contains('Бесплатно!')").remove();

    Here's a JSFiddle demonstrating this.

    If you're not looking to delete the entire node (it's not so clear from your question), then $("li:contains('Бесплатно!')") is a jQuery object containing all of the elements whose children you're looking to delete elements from. This would make it easier for you to find the appropriate <li> children and remove them from the DOM.

    document.querySelector('.star-rating')

    will give you only first matching node.

    You should use document.querySelectorAll which will return a node list

    var x = document.querySelectorAll('.star-rating');
for(var i in x){
    x[i].parentNode.removeChild( x[i].nextSibling );
}
