    Градиенты для текста

    • 0
    • 1
    • 2
    • 3
    • 4
    • 5
    Написал: James DGreeze, 20-09-2011, 17:42 | Просмотров: 872 | Комментариев: 0 | Рубрика: JavaScript
    Многие из вас наверно ищут способы реализации текстовых градиентов. В CSS это пока нельзя сделать. только блоки и то не во всех браузерах.
    Сегодня я покажу как сделать это все быстро, удобно и без лишних телодвижений.
    ДемоИсходники
    Для этого мы с вами будем использовать библиотеку jQuery FontEffect Plugin.

    Это замечательный плагин, который я использую в своих проектах для того, чтобы создавать градиенты и зеркальные отражения текстам

    Итак, ПОЕХАЛИ!


    Сделаем сегодня вот такой градиент:


    И начнем мы с того, что разберем настройки библиотеки и узнаем, какие возможности она нам дает.

    Первое что нам надо это подключить библиотеку, что мы сделаем засунув в ее хедер страницы ниже! библиотеки jQuery иначе не будет работать.
    <script type="text/javascript" src="/jquery-FontEffect-1.0.0.min.js"></script>


    затем давайте посмотрим настройки:

    $(Obj).FontEffect({
        outline         :false,    // Обводка
        outlineColor1   :"",       // Верхняя граница обводки
        outlineColor2   :"",       // Нижняя граница обводки
        outlineWeight   :1,        // ширина обводки  1=тонкая,2=средняя,3=жирная
        mirror          :false,    // Зеркальность
        mirrorColor     :""        // Цвет зеркального отображения
        mirrorOffset    :-10,      // Расстояние от осн. текста
        mirrorHeight    :50,       // Высота отражения
        mirrorDetail    :1,        // Детализация 1=Высокая,2=Средняя,3=Низкая
        mirrorTLength   :50,       // Тоже какая то длина :)
        mirrorTStart    :0.2,      // Непрозрачность от 0 до 1
        shadow          :false,    // Тень
        shadowColor     :"",       // Цвет тени
        shadowOffsetTop :5,        // Напрвление по оси X
        shadowOffsetLeft:5,        // Напрвление по оси Y
        shadowBlur      :1,        // Размытие 1=нет,2=слабое,3=сильное
        shadowOpacity   :0.1,      // Непрозрачность от 0 до 1
        gradient        :false,    // Градиент
        gradientColor   :"",       // Цвет градиента
        gradientPosition:20,       // Начальная позиция
        gradientLength  :50,       // Длина градиента
        gradientSteps   :20,       // Размер шага
        hideText        :false,    // Скрывает осн. текст
        })


    Итак, с опциями разобрались
    Осталось последнее, назначить тексту эффекты.
    Это мы сделаем через такой код
    $('#НАЗВАНИЕ ЭЛЕМЕНТА').FontEffect({
        //тут прописываем те опции которые будем использовать 
    })


    Для нашего примера мы применим следующий код:

    $('#demiart').FontEffect({
        outline         :true,
        outlineColor1   :"#fffaf3",
        outlineColor2   :"#524125",
        outlineWeight   :1,
        mirror          :true,
        mirrorColor     :"#fffaf3",
        mirrorOffset    :-25,
        shadow          :true,
        shadowColor     :"#4f4627",
        shadowOffsetTop :0,
        shadowOffsetLeft:0,
        shadowBlur      :2,
        shadowOpacity   :0.5,
        gradient        :true,
        gradientColor   :"#fdfaeb",
        gradientPosition:20,
        gradientLength  :40,
        gradientSteps   :20
    })


    А стили будут такие:
    CSS

    body {
        background-color: #1f1a17;
        width: 100%;
    }
    h1#demiart {
        width: 200px;
        font-size: 90px;
        font-family: Georgia;
        font-weight: normal;
        margin: 30px auto 0;
        color: #4f4627;
    }


    Вот и все! Если есть вопросы оставляйте в комментариях отвечу.

    Небольшой PS: При использовании данного плагина, перестает работать такое CSS свойство как text-align:center;
    Поэтому имейте ввиду и не удивляйтесь что ваше центрированое лого после применения градиента имеет float:left
    Все это лечится установлением width на тег и margin: auto на лево и право).
