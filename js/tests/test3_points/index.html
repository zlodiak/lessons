Игра Нервное Зрение

Время: Очки:

Правила игры

  1. заполнять игровое поле цветными квадратиками (целями), нажимая соответствующую кнопку
  2. набирать очки нажатием на цветные квадратики с цифрами (цели)
  3. запоминать цвета уничтоженных целей. если цвет уничтожаемой в данный момент цели совпадает с цветом уничтоженной 2 хода назад, то количество очком уменьшается на 20
  4. при уничтожении цели с цифрой 0 очки не прибавляются, но на игровое поле добавляется 20 новых целей
  5. цвет игрового поля меняется случайным образом
