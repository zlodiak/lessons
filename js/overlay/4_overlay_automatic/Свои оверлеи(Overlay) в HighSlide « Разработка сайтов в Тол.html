Свои оверлеи(Overlay) в HighSlide


Встала передо мной задача организации галереи товаров. Причем, по желанию заказчика, она должна быть во всплывающем окне. Не долго думая, я решил использовать довольно мощный и гибкий лайтбокс – HighSlide. С помощью него получилось организовать галерею с перелистыванием изображений практически безо всяких усилий.

Ссылка на пример
Официальный сайт HighSlide (В разделе «Tutorial» можно найти подробный учебник)

Но тут у заказчика появилось желание разместить над этой фотографией (то есть в развернутом виде) пару кнопок, вроде «Заказать» и «Информация». Они должны быть функциональны и выполнять свою функцию. Реализацию корзины магазина и высплывающего окна я показывать не буду, а расскажу, как средствами HighSlide добавить такие кнопки на фотографию. Бегло просмотрев ресурсы рунета найти инструкцию на русском так и не удалось, так что решил написать свою.

Есть такой полезный инструмент в этой галерейке, как оверлеи (суть, как в API Яндекс-карт).

Теорию расписывать не буду, лучше сразу перейдем к практике.

Само собой, сначала подключим js и css галереи.

<link rel="stylesheet" type="text/css" href="highslide/highslide/highslide.css">
<script type="text/javascript" src="highslide/highslide/highslide-full.packed.js"></script>

Вставим в разметку несколько изображений и инициализируем галерею.

<div id="hte-gallery">
 <a href="images/1.jpg" onclick="return hs.expand(this)" id="dd">
 <img src="images/1s.jpg" title="Природа">
 </a>

 <a href="images/2.jpg" onclick="return hs.expand(this)">
 <img src="images/2s.jpg" title="Природа">
 </a>

 <a href="images/3.jpg" onclick="return hs.expand(this)">
 <img src="images/3s.jpg" title="Природа">
 </a>
 </div>

		<script type="text/javascript">
			//Инициализация галереи
			hs.graphicsDir = 'highslide/highslide/graphics/';
			hs.align = 'center';
			hs.transitions = ['expand', 'crossfade'];
			hs.wrapperClassName = 'dark borderless floating-caption';
			hs.fadeInOut = true;
			hs.dimmingOpacity = .75;
</script>

На данный момент изображения просто будут разворачиваться. Теперь добавим встроенный таскбар, с помощью функции hs.addSlideshow, с основными элементами управления галерей.

		<script type="text/javascript">
			//Инициализация галереи
			hs.graphicsDir = 'highslide/highslide/graphics/';
			hs.align = 'center';
			hs.transitions = ['expand', 'crossfade'];
			hs.wrapperClassName = 'dark borderless floating-caption';
			hs.fadeInOut = true;
			hs.dimmingOpacity = .75;

			//Добавляем стандартный таскбар управления изображениями
			if (hs.addSlideshow) hs.addSlideshow({
				interval: 5000,
				repeat: false,
				useControls: true,
				fixedControls: 'fit',
				overlayOptions: {
					opacity: .6,
					position: 'middle center',
					hideOnMouseOut: true,
				}
			});
</script>

Теперь подошло самое время разместить простейший оверлей над нашей галереей.
Вставим, например, div с названием тематики галереи, в данном случае «Природа».

<div id="overlay1" class="overlay1">
Природа
</div>

// Добавляем простейший оверлей для галереи
			hs.registerOverlay({
				overlayId: 'overlay1',
				position: 'top left',
				hideOnMouseOut: true,
				opacity: 0.8,
				classname: 'overlay1',
				offsetX: 20,
				offsetY: 20,
				fade: 0,
				relativeTo: 'viewport'
			});

В случае, если нужно разместить несколько оверлеев, можно вызывать функцию hs.registerOverlay сколько угодно раз. Если нужно создать разные оверлеи для разных изображений, то здесь уже придется группировать изображения в отдельные галереи с помощью параметра slideshowGroup, либо использовать шаблоны внутри оверлея, но это уже выходит за рамки ознакомительной статьи с оверлеями.

Здесь приведу пояснения к параметрам функции hs.registerOverlay:

className: Имя класса для оверлея. Это имя класса CSS, которое будет применено непосредственно к оверлею.

fade: Использование плавного появления или исчезновения. Может принимать три значения: 0 – не использовать, 1 – использовать, 2 – использовать только при отсутствии в IE использования альфа-фильтров для PNG-прозрачности (в ином случае, могут всплывать нежелательные артефакты).

hideOnMouseOut: Флаг показывает, прятать или оставлять оверлеи поверх изображения при потере изображением фокуса мышки.

html: Строка html, которая будет размещена внутри нового оверлея. При использовании данного свойства использовать свойство overlayId не нужно.

overlayId: Идентификатор блока, который будет использоваться в качестве оверлея.

offsetX, offsetY: Смещение оверлея относительно текущего положения в пикселях. Может принимать отрицательные значения.

opacity: прозрачность оверлея. Значения от 0 до 1 с плавающей точкой.

position: Указывает положение, где появится оверлей. Первое значение – вертикальное положение и может принимать значения: above, top, middle, bottom или below, – второе слово – горизонтальное положение и может принимать значения: left, center, right. Также свойство может принимать значение в виде одного слова leftpanel или rightpanel. Также можно использовать выражения margin в CSS для задания положения этого блока.

relativeTo: Может принимать два значения: ‘viewport’ или ‘expander’. В случае, если стоит первое значение – положение оверлея высчитывается относительно всей области галереи (включая затемнение), если же второе – то относительно области изображения.

slideshowGroup: указывает на идентификатор группы изображений, в которой будет присутствовать данный оверлей.

thumbnailId: Любая строка, принимающая идентификатор с миниатюрами, или значение null для наложения на все миниатюры.

width: Ширина наложения. Может быть задана либо в пикселях либо в процентах. Тем не менее позиции leftpanel и rightpanel требуют фиксированной ширины в пикселях, или по умолчанию для них задается ширина 200px.

useOnHtml: По умолчанию оверлеи не добавляют HTML контент. Поставить эту опцию для добавления.

Метки: highslide, lightbox, overlay, лайтбокс, оверлеи, оверлей
