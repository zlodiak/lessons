                      alanstorm Mod Greg • 3 years ago

                      Greg, thank you for the bug report — not enough people take the time so it's appreciated.

                      The problem was a typo in my naming of the controller class file. Magento wanted the class name `Pulsestorm_Adminhello_AdminhelloController`, but I accidentally camel cased the filename to `AdminHelloController.php`. The capitol `H` in hello threw off the controller loader, which only lead cases the `A`. I missed this in QA since I'm on a Mac, which ships with a "sort of" case sensitive file system.

                      I've fixed the article and the archive so it should work now with the `adminhtml/adminhello/index` URL/Route.
