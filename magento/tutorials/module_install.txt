как установить модуль:

Отключите компиляцию, если включена (Система -> Инструменты -> Компиляция)
Отключите кэш, если включён (Система -> Управление кэшем)
Скачайте файл архива или установите модуль с Magento Connect
Если вы скачали архив, то скопируйте файлы из папки "install" в корневую папку сайта - где находится файл index.php
Откройте доступ к сайту. Для это вам необходимо удалить (переименовать) файл maintenance.flag
Выйдите из панели администрирования (Выйти)
Войдите в панель администрирования (введите логин/пароль)
Настройте параметры модуля (Система -> Конфигурация -> ET EXTENSIONS -> IP Security)
Если необходимо запустите процесс компиляции и включите кэш
