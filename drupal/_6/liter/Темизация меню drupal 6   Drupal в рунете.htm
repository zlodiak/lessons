А вот в 6м, так и не понял как делать.

Прочитав данные темы:
http://www.drupal.ru/node/30209
http://www.drupal.ru/node/30727

т.е. создаю файл block-menu-primary_links.tpl.php или block-menu-primary-links.tpl.php

<?php

  foreach ($nav as $item) {
    print '<a href="'. $item['href'] .'" class="menu1">'. $item['title'] .'</a><br><br>';
  }

Делаю по другому, в template.php переопределяю функцию phptemplate_links(), изменений никаких...

не знаю в какую сторону рыть уже.

заранее спасибо.

Сам разобрался. Если кому-то пригодится - вот код.

<ul class="menu_ul_class">
<?php
  $nav = menu_navigation_links('primary-links');
  foreach ($nav as $item) {
  	$href = $item['href'] == "<front>" ? base_path() : base_path() . drupal_get_path_alias($item['href']);
    print '<li class="menu_li_class"><a href="'. $href .'">'. $item['title'] .'</a></li>';
  }
?>
</ul>
