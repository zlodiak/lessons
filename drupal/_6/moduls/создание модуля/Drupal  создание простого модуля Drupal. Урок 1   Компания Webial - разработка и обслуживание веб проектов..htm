Drupal: создание простого модуля Drupal. Урок 1

Опубликовано Assembler в Втр, 03.05.2011
  • CMS

Данная статья представляет собой краткое пошаговое руководство по разработке простого Helloworld модуля в Drupal 6.x, который выполняет одну простую функцию – вывод на экран надписи "Hello World".

Для этого Вам необходимо выполнить такие действия:

1. Создайте новую папку для вашего модуля

В каталоге yoursite/modules создайте папку с названием Hellow. Данная папака предназначена для хранения основных модулей сайта, в том числе и нашего нового модуля.

2. В новом каталоге создайте файл с расширением *.info: в данном случае файл будет иметь название hellow.info

Содержание файла hellow.info :

; $Id$
    name = Module  Hello World
    description  = My first Drupal 6.x Module
    core = 6.x

Этот файл позволяет Drupal опознать наш модуль в системе.

3. Создайте файл с расширением *.module:  в нашем случае файл будет иметь название hellow.module.

Мы должны указать для Drupal путь, под которым может быть доступен наш модуль. Делается это с помощью функции hook_menu (). Для обрабоки всех событий в модуле необходимо объявить функцию: modulename_hookname (). В нашем случае она будет выглядеть как hellow_menu () {}. Данная функция описана ниже:

function hellow_menu(){
$items = array();
$items['hellow'] = array(
'page arguments' => array(1),
'title' => t('Hello World - Module'),
'page callback' => 'hellow_page',
'type' => MENU_CALLBACK,
'access arguments' => array('access content'),
  );
  return $items;
}

Мы создали массив $items['hellow'], который позволяет создать определенные настройки для нашего модуля в системе.

  1. title: заголовок страницы.
  2. page callback: это функция, которая вызывается при открытии ссылки на модуль.
  3. page arguments: количество аргументов, которые возвращаются при вызове.
  4. access arguments: массив прав доступа. В данном случае "access content" - абсолютный доступ к странице.
  5. type: определяет проверку hook_menu ().

Таким образом, при открытии ссылки /yoursite/hellow произведется обратный вызов функции hellow_page ().

Теперь создадим hellow_page () функцию, содержание которой выглядит следующий образом:

function hellow_page($argument) {
   return "Hello World!!!";
}

Итак, общий вид файла hellow.module:

<?php
function hellow_menu(){
$items = array();
$items['hellow'] = array(
'page arguments' => array(1),
'title' => t('Hello World - Module'),
'page callback' => 'hellow_page',
'type' => MENU_CALLBACK,
'access arguments' => array('access content'),
  );
  return $items;
}
function hellow_page($argument) {
    return "Hello World! Arguments parsed was " . $argument;
}

Теперь перейдите в Admin/build/modules и подключите наш модуль. Доступ к модулю можно получить по ссылке //yoursite.com/hellow
