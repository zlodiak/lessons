Программный вывод блока в Drupal 7

05/07/2012 | Блог

Для программного вывода(в коде темы, вне региона) содержимого любого блока в Drupal 7 используйте конструкции следующих видов:

Для вывода пользовательского блока

<?php
$block = block_load('block', 1); // id блока
$output = drupal_render(_block_get_renderable_array(_block_render_blocks(array($block))));
print $output;
?>

 

Для вывода блока предоставляемого модулем

<?php
// Выводим блок поиска по сайту
$block = module_invoke('search', 'block_view');
print render($block['content']);
?>

 

Для вывода блока из Views

<?php
$block = module_invoke('views', 'block_view', 'ИМЯ_БЛОКА-block');
print render($block['content']);
?>
