Свой отдельный шаблон страницы page для типа материала node type drupal

Здравствуйте, уважаемые читатели блога Stanem.pro!

Для того, чтобы была возможность темизировать шаблон для конкретного типа материала в cms drupal, нужно ввести следующий код в файл template.php, который находится в папке с вашей темой

function ТЕМА_preprocess_page(&$vars, $hook) {
    if (isset($vars['node'])) {  
        $vars['theme_hook_suggestions'][] = 'page__'. str_replace('_', '--', $vars['node']->type);
    }
}

Вместо слова "ТЕМА" введите название используемой темы на латинице с маленькой буквы, например, bartik.

Теперь, когда вы добавили код, можно создать отдельный шаблон для типа материала.
Нужно создать файл page--тип.tpl.php, где вместо слова "тип" нужно вписать название типа материала, например, news. И добавить нужный вам код.

После этого нужно обязательно сбросить кэш по ссылке /admin/config/development/performance.
