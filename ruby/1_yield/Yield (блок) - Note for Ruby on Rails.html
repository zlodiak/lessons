Yield (блок)

Блок - один из способов использования замыканий в Ruby. Блоком является выражение внутри {} если это одно строчный блок или выражение между do..end.
yield - (переводится как собрать) буквально означает: вместо этого слова в методе нужно поставить блок кода (убрав фигурные скобки) и выполнить метод в обычном режиме.
Вызов метода осуществляется: пишется название метода и блок кода в качестве параметра.
Пример
def test1 # yield без параметра
    a = 1
    yield
end

#вызов метода
> test1{puts "Hello!"} #  Hello!

#параметр в блоке не указан
Для того чтобы метод  test1 выводил именно переменную а, нужно yield передавать параметр (в нашем случае а) и этот параметр сразу же будет использоваться блоком при замене yield на блок.
Пример (параметр из yield передается в блок)
def test2 # yield и параметр
    a = 1
    yield a
end

>test2{|x| puts ((2 + x).to_s + 'hi')}
Чтобы вызвать метод у которого yield с параметром: пишется имя метода и параметром в блоке между  | | перечисляются параметры, которые которые будут браться у yield
Вертикальные скобки, которые обрамляют параметры называются pipes (трубы). Блок возвращает результат того, что идет после  pipes.
В примере |x| означает: при замене yield  на блок - взять параметр у yield  и подставить его в x.
Порядок действий:
  • Вместо yield подставить блок кода без фигурных скобок.
  • Взять параметр у yield (а в нашем случае) и передать его х.
  • Выполнить метод в обычном режиме.
В итоге предыдущий пример будет выполнятся как если бы мы ее написали вот так
Пример
def test2
    a = 1
    puts ((2 + 1).to_s + "hi")
end
Если замыкание обязательно, следует поместить его, как последний аргумент метода и вначале аргумента добавить амперанд:
def test(&closure)
    return a if a
    yield "и раз"
    yield "и два"
end

>test true   # true
>test false  # Local Jump Error: no block given
Поэтому, лучше вместо введения обязательного параметра задать замыкание по умолчанию. Запуск блока кода на выполнение производится обращением к методу call этого объекта т.е. .call   - явный способ вызова замыкания на выполнение.
Пример
def test(n, m, &blk)
    for i in n..m
        blk.call(i)
    emd
end

>test(3, 5){|i| print "#{i} ... " }

# '3...4...5...'
