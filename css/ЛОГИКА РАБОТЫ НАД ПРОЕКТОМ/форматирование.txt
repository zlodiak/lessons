форматирование css:
1. отступы
	делать для всех селекторов(кроме html и body)
	делать только для основных селекторов (header, content, footer и т.п.)
	не делать
	в случае перечисления подобных селекторов через запятую помещать их столбиком
2. группы селекторов, порядок селекторов в группе
	глобальный ресет/прочие блоки/основные блоки (внутри групп селекторы в порядке следования). основные блоки помещать последним пунктом, так удобнее для быстрого перехода
	без групп, селекторы в порядке следования
3. группы свойств
	упорядочить по алфавиту
	упорядочить по длинне строки
	разбить на логические группы(группа свойств позиционирования, группа свойств размеров элемента и т.п.)
	не упорядочивать
4. комментарии
	комментировать группы селекторов(если есть)
	комментировать комментировать основные блоки(header, content, footer и т.п.)	
	поместить в начале css-файла список основных блоков страницы(возможно с указаннием соответствующего класса)
	не комментировать
5 .модель блока селектора
	однострочный блок(свойства отбивать табуляцией)
	многострочный блок
6 .хлебные крошки селектора
	всегда указывать полный набор селекторов(кажется такой способ вызывает тормоза в браузере opera)
	указывать только необходимый селектор
7. именование блоков
	всегда через class
	основные блоки(header, content, footer и т.п.) через id, остальные через класс
