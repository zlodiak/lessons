Re: Редирект после выхода.

Можно добавить фрагмент в functions.php активной темы:

Код:

function redirect_after_logout($logout_url, $redirect) {
    if ( empty($redirect) )
        $logout_url = add_query_arg('redirect_to', urlencode('адрес нужной страницы'), $logout_url);

    return $logout_url;
}
add_filter('logout_url', 'redirect_after_logout', 10, 2);
Успехов!
