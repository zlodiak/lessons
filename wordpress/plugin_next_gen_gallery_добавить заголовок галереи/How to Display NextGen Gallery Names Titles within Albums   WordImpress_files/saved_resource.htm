  • Andreas Larsson

    Messing with the core files should be avoided if possible. Instead, write your own function in functions.php and use the nggdb class to fetch whatever you need. That way, your modifications will stay intact even if you update the plugin in the future. And you will have better control over your code and the plugin developer's. If you need to modify the code 6 months from now, you will find it instantly instead of searching through the core files. 

    Look through nextgen-gallery/lib/ngg-db.php and you will get an idea of what you can do. Use Wordpress' add_shortcode function to create shortcodes to your own nextgen functions and it will be super simple to insert your customized galleries through wp-admin.
