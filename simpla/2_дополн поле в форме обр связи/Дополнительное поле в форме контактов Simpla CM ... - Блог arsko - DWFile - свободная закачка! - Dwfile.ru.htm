Дополнительное поле в форме контактов Simpla CM ...

Топ 2Читали блог: 458 Создано: 2012-3-8 21:45 |Личные категории:simpla

Недавно стал осваивать новую для себя CMS под названием Simpla. По сути это скрипт интернет-магазина, простой и в то же время мощный.
Итак, начну!
Делаем дополнительные поля в форме контактов. Все просто! Начнем с правки файла feedback.tpl вашего шаблона в папке /www/yourdomain.ru/design/yourdesign/html . В форме вставляем нужное вам поле, в нашем случае поле номер телефона:                <label>Номер телефона</label>

                <input data-format=".+" data-notice=" Введите номер телефона " value="{$ phone|escape}" name=" phone" maxlength="255" type="text"/></td>

Если нужно сделать данное поле обязательным, чуть выше поправим код вывода ошибок:

                <div class="message_error">

                               {if $error=='captcha'}

                               Неверно введена капча

                               {elseif $error=='empty_name'}

                               Введите имя

                               {elseif $error=='empty_email'}

                               Введите email

                               {elseif $error=='empty_phone'}

Введите номер телефона, ну пожалуйста =(

                               {elseif $error=='empty_text'}

                               Введите сообщение

                               {/if}

                </div>

Далее правим файл FeedbackView.php в папке /view/

Добавляем строку

 $feedback-> phone       = $this->request->post('phone');

Чуть ниже строку

$this->design->assign('phone ', $feedback-> phone);

А так же вывод ошибки пользователю

elseif(empty($feedback-> phone))

                $this->design->assign('error', 'empty_ phone');

Теперь сделаем маленькое изменение в таблице базы данных s_feedbacks, вставим пустую строку phone с параметрами: varchar, Null Нет.

Все, закончили с фронтэндом, теперь переходим к админке.

Для нормальной работы дизайна админки создаем ID comment_tel в style.css

Вставляем его.

div.comment_tel{

                font-size: 12px;

                color: #000000;

}

Я взял тупо ID текста сообщения, время 2:36 ночи не до работы уже, но я обещал.

Далее в файле feedbacks.tpl, папки /simpla/design/html/ вставляем код

<div class=' comment_tel '>

{$feedback->phone|escape}

</div>

после данной строки

<a href="mailto:{$feedback->name|escape}<{$feedback->email|escape}>?subject=Вопрос от пользователя {$feedback->name|escape}">{$feedback->name|escape}</a>

И последнее, для того чтобы заработало поле вывода номера телефона в админке, редактируем файл Feedbacks.php в папке /api

Строка 17, после

                public function get_feedback($id)

                {

меняем

$query = $this->db->placehold("SELECT f.id, f.name, f.email, f.ip, f.message, f.date FROM __feedbacks f WHERE id=? LIMIT 1", intval($id));

На

$query = $this->db->placehold("SELECT f.id, f.name, f.email, f.ip, f.phone, f.message, f.date FROM __feedbacks f WHERE id=? LIMIT 1", intval($id));

Теперь у нас есть поле ввода номера телефона и отображается в админке.
