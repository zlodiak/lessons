Установить VMware Player 5.0.2 для Linux в Ubuntu 13.04/12.10/12.04 и Linux Mint 14/13


VMware Player — бесплатный программный продукт для некоммерческого использования, на основе виртуальной машины VMware Workstation, но с ограниченной функциональностью, предназначенный для запуска образов виртуальных машин, созданных в других продуктах VMware.

VMware Player для Linux был обновлён до версии 5.0.2.

Ниже приведены некоторые из изменений, которые были сделаны в этой версии:

При установке виртуальных машин VMware Tools в Linux, параметры разрешения экрана по умолчанию были стандартизированы до 640 × 480, 800 × 600, 1024 × 768 или 1280 × 800 для первоначального входа в систему. После авторизации, разрешение может быть установлено на любое значение по желанию пользователя или может быть использован Автоподбор.
Список доступных сетевых адаптеров в автоматическом режиме, в диалоговом окне Настройка, теперь обновляется без перезагрузки Player'а.
При работе в Linux, поддерживает среду рабочего стола GNOME 3.
VMware Tools больше не падает на операционной системе Windows 2008 Server Core x64 SP2, со следующей ошибкой:
  VMware Tools неисправимая ошибка: (vthread-3)
  Исключение 0xc0000005 (нарушение прав доступа) не произошло.
Тем не менее, Windows 2008 Server Core x64 SP2 не поддерживается операционной системой VMware Workstation.

Полный список изменений об этом релизе, на странице: VMware Player 5.0.2 Примечания к выпуску.
   
Установка VMware Player 5.0.2

Прежде всего нужно загрузить пакет VMware-Player-5.0.2-bundle по одной из ссылок, согласно архитектуры вашего компьютера ОТСЮДА.

После загрузки файла убедитесь, что пакет находится в папке Загрузки:



Если файл находится в другой папке, перенесите его в папку Загрузки и выполните команду в терминале для перехода в папку:

cd ~/Загрузки

Далее выполните следующую команду, чтобы сделать загруженный файл исполняемым:

chmod +x VMware-Player-5.0.2-1031769.i386.bundle

И последнее. Выполните команду для установки VMware Player:

sudo ./VMware-Player-5.0.2-1031769.i386.bundle

Следуйте указаниям мастера до завершения установки.

По окончании установки, вы найдёте VMware Player 5.0.2 в программах Системные утилиты или поиском в меню Даш:




Примечание. Хочу предупредить, что программа на английском языке, но не думаю, что у вас возникнут трудности.

Удаление VMware Player

Если вы хотите удалить VMware Player из системы, выполните следующую команду:

sudo vmware-installer -u vmware-player

Удачи.
