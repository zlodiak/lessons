Установка LAMP-сервера на Linux Mint 15/16/17

02.11.2013 / Posted in  Browser, How To, Linux, Mac, Web, Wordpress

LAMP-сервер, это полнофункциональный WEB-сервер, включающий в себя Linux, Apache, MySQL и PHP. Отсюда и идет название – LAMP. Этот сервер нужен для локального тестирования сайтов и последующей выгрузкой из на хостинг или даже для использования своейго компьютера в качестве сервера без хостинга. Но для этого необходимо иметь белый IP-адрес и возможность подключения доменного имени.

Для начала надо перейли в интерактивный режим sudo (sudo -s или sudo -i) и установить Apache. После установки при заходе на 80-ый порт через любой браузер на localhost или IP-адрес машины будет выводиться соббщение, что сервер Apache готов к работе – It Works!

1
2
sudo -i
apt-get install apache2

После этого нам надо установить PHP и различные модули для него. На данный момент будет установлен простой PHP, необходимые модули можно потом уже поставить самому.

1
apt-get install php5 libapache2-mod-php5

Для проверки установленного PHP можно вывести phpinfo. Для этого делаем :

1
echo '<?php phpinfo(); ?>' > /var/www/phpinfo.php

И после этого заходим на IP-адрес или на localhost, указывая phpinfo.php.

Теперь установим MySQL сервер. Во время установка MySQL сервера необходимо будет ввести пароль root’а для сервера MySQL.

1
apt-get install mysql-server

Теперь проверяем :

1
mysql -u root -p

Выходим (exit) и устанавливаем дополнительные модули. Если потребуется пароль, а он обязательно потребуется – надо ввести установленный пароль от MySQL. PhpMyAdmin будет доступен в http://localhost/phpmyadmin или в http://ip-adress/phpmyadmin.

1
apt-get install libapache2-mod-auth-mysql php5-mysql phpmyadmin

И не забываем добавить строчку

1
Include /etc/phpmyadmin/apache.conf в файл /etc/apache2/apache2.conf

Сюда

1
nano /etc/apache2/apache2.conf

Теперь можно не волноваться – сервер полностью готов к использованию.
