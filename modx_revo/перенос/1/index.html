Ошибка Error 503 Site temporarily unavailable при переносе сайта на хостинг modx Revolution

При переносе сайта на ModX Revolution с локального сервера на другой локальный сервер либо на хостинг возникает ошибка Error 503 Site temporarily unavailable

  1. Сайт — обновить сайт (очистка кеша) (mysite\core\cache)
  2. Изменить пути в 4 файлах config.core.php находящихся по следующим путям
  • mysite/config.core.php
  • mysite/manager/config.core.php
  • mysite/connectors/config.core.php
  • mysite/core/config/config.core.php
