cli backup

===============================
Backups are stored in the var/backups directory and can be restored at any time using the magento setup:rollback command.

===============================
вывести список всех бекапов:
./bin/magento info:backups:list


===============================
сделать бекап, который будет состоять из файлов фреймворка, статики, дампа БД:
bin/magento setup:backup --code --media --db

===============================
пример отката: 
magento setup:rollback [-c|--code-file="<name>"] [-m|--media-file="<name>"] [-d|--db-file="<name>"]
For example, to restore a media backup named 1440611839_filesystem_media.tgz, enter
magento setup:rollback -m 1440611839_filesystem_media.tgz
