PIL - подгружаемая библиотека для расширенной работы с изображениями.

PILLOW - более современнаяверсия. примечательно, что в скриптах подключается так же. то есть import PIL

PILLOW соодержит 2 важных класса для работы с изображениями:
Image - позволяет обрабатывать изображения(переворачивать, менять размер, получать инфо, сохранять и т.д.)
ImageTK - позволяет выводить изображения на экран. расширяет возможности встроенной в питон библиотеки для работы с изображениями. в частности встроенная библиотека может работать только с форматом gif. а ImageTK с jpg, png и множеством других.

подключатся примерно так 
from PIL import Image, ImageTk
