Чем открыть и отредактировать базу данных SQLite

Категория: Софт-обзоры
10 апреля 2012

В одной статье я писал, что куки в Mozilla Firefox хранятся, во-первых, не в нескольких файлах, как это бывает с другими браузерами, а в одном. Во-вторых, файл, в котором хранятся cookies, являет из себя не текстовый файл, а обычную базу данных формата SQLite. Этот формат очень часто используется разными программами для хранения своих данных. К примеру, менеджер буфера обмена Clipdiary тоже использует базу данных формата SQLite для сохранения в нём истории данных, которые пользователь копирует в буфер обмена. Так что sqlite - это очень популярный формат баз данных (правда на серверах он используется редко, если Вам это о чём-то говорит).

Чем же можно открыть и отредактировать эту базу данных? На самом деле, программ для этого много, я всего лишь расскажу о двух из них.

SQLite Studio

Программа от польских разработчиков, скачать можно с официального сайта. Программа кросплатформенная, не требует установки, имеет простой и понятный интерфейс (хотя и на английском; ну что поделаешь, английский язык сейчас уже даже не считается иностранным; кстати, Вы знаете, что на этом сайте есть раздел, в котором размещены кроссворды на английском языке?).

При запуске программа начинает показывать разные советы, можно просто закрыть окно кнопкой “Close”.

Для того, чтобы открыть базу данных SQlite, выберите в меню “Databases” пункт “Connect” (или “Add Database”). Откроется меню добавления файла, в котором надо указать расположение нашего файла. В случае с куками браузера Мазилла Фаерфокс, надо выбрать файл cookies.sqlite в папке профиля пользователя. Присоединившись к базе данных, мы увидим таблицы, и в меню Tables надо выбрать интересующую нас таблицу, в нашем случае её имя - moz_cookies. Чтобы отредактировать значения данных в нашей базе, надо перейти на вкладку Data и просто кликнуть на нужном значении, подобно тому, как редактируются значения ячеек Microsoft Office Excel.

Более подробно говорить об этой программе нет смысла, так как если уж человек собрался редактировать базу данных sqlite, то он наверняка знает, что он делает, и ему не составит труда разобраться с SQLite Studio.

SQLite Database Browser

Ещё одна программа, которой можно открыть, посмотреть и изменить базу данных SQLite - это программа SQLite Browser. SQLite Browser простой в работе, наверное, ещё проще, чем SQLite Studio. Однако, она по какой-то причине отказалась открывать файл базы данных куков Mozilla Firefox, однако успешно работала с другими файлами баз данных. В чём причина, я не знаю, может быть дело в том, что SQLite Browser немного устарел, а формат развивается.

SQLite Manager (расширение для Firefox)

Есть ещё расширение для Firefox под названием SQLite Manager. Несмотря на то, что это дополнение для известного браузера, оно может открывать и изменять не только SQLite-файлы из этого браузера, но и любые другие базы. SQLite Manager имеет интерфейс, аналогичный с двумя предыдущими, но в довесок имеет русский интерфейс. Сама программа вызывается из меню Mozilla Firefox «Инструменты» (для того, чтобы отобразить меню в новых версиях Mozilla Firefox, можно нажать клавишу Alt).

Надеюсь, что эта статья дала исчерпывающий ответ на вопрос, чем можно изменить базу данных SQLite. Естественно, описанные здесь средства не являются единственными, но их вполне достаточно для выполнения требуемой задачи.
